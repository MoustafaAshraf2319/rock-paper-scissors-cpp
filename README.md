# Stone, Paper, Scissors Game (C++)

A robust, console-based implementation of the classic **Stone, Paper, Scissors** game, developed as part of my C++ learning journey. This project focuses on applying clean code principles, structured programming, and logical game flow.

## 🚀 Key Features
- **Dynamic Rounds:** Players can choose between 1 to 10 rounds per game.
- **Smart Logic:** Automated computer opponent using randomized choices.
- **Visual Feedback:** Interactive screen colors (Green for win, Red for loss, Yellow for draw).
- **Comprehensive Statistics:** Detailed summary after each round and a final game result report.

## 🛠️ Concepts Applied
In this project, I demonstrated my understanding of:
- **User-Defined Data Types:** Using `struct` to organize round and game data.
- **Enums:** For managing game choices and winner states clearly.
- **Functions & Modularization:** Splitting logic into small, reusable functions (Read, Logic, Print, Reset).
- **Arrays & Mapping:** Efficiently converting Enums to readable strings.
- **Loops & Control Flow:** Managing game state and "Play Again" functionality.

## 📋 Requirements
- Any C++ compiler supporting C++11 or higher.
- Windows OS (Required for `system("color")` and `system("cls")`).
